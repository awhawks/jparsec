Provides classes for applying the old IMCCE theories VSOP87, Series96, and ELP2000 Lunar Solution.
This package contains an implementation of the old theories VSOP87, Series96, and of ELP2000 Lunar Solution by the IMCCE. Series96 and VSOP require external .jar files in the classpath to be used, but Elp2000 data in included in this package.
